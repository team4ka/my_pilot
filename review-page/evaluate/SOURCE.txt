Generate index.html from the project root:
  python capture_evaluate.py
  (automatically runs scripts/static_evaluate_snapshot.py — removes Next/analytics/reCAPTCHA

  python scripts/static_evaluate_snapshot.py

By default a visible Chromium window opens (HEADLESS=0). You get ~18s to click
“Alle akzeptieren” if the auto-click did not close the banner (override with COOKIE_MANUAL_SEC).

Before saving, the cookie banner markup is removed from the DOM so the snapshot is clean.

HEADLESS=1              — no window (CI / automation)
COOKIE_MANUAL_SEC=0     — skip the manual cookie wait when headed
